# time-series-lstm-shap-forecasting
# Advanced Time Series Forecasting with LSTM & SHAP Explainability

This project implements a multivariate time series forecasting model using:
- Long Short-Term Memory (LSTM)
- Sliding window preprocessing
- Hyperparameter tuning
- SHAP explainability for feature/time-step contribution

## Contents
- Python code (LSTM model)
- Hyperparameter search
- SHAP Explainability
- Full project report
- Dataset generation script

## How to run
Install dependencies:
pip install numpy pandas scikit-learn tensorflow shap matplotlib

Run the script:
python time_series_lstm_shap_project.py
